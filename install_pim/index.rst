Deploy Akeneo PIM
=================

You can use `Capistrano <https://github.com/capistrano/capistrano>`_ and its `Symfony plug-in <https://github.com/capistrano/symfony>`_ to deploy Akeneo.

You can follow :doc:`this step-by-step guide </install_pim/deploy/capistrano>` to set it up.
